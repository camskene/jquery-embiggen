jQuery Embiggen Plugin

The jQuery Embiggen plugin will display any linked image center to the window.

A semi-transparent overlay is displayed behing the image.

A spinner is displayed while the image is loading.

If the image is larger than the viewport it will, by default, be scaled proportionally to fit.

Clicking the embiggened image or overlay closes both.

Download

Download from Github.

Options

There are 3 options.

                    vertical: true,
                    bgcolor: "#191919",
                    opacity: "0.98"
  1. Set vertical to false to scale the width of the image only to the window. This is handy if you have an image with a large height. The image is always scaled proportionally.
  2. Set bgcolor to desired overlay color.
  3. Set opacity to desired overlay opacity.

Setup

HTML

                <a href="img/lg/biy-home-001.jpg">
                    <img src="img/th/biy-home-001.jpg" width="630" height="492" />
                </a>

JS

                $("a").embiggen();

or

                $("a").embiggen({
                    vertical: true,
                    bgcolor: "#191919",
                    opacity: "0.98"
                });

Demo I

                $("a").embiggen();

Demo II

                $("a").embiggen({
                    vertical: false
